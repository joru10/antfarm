# News Agent

Fetches and verifies news from multiple sources.

## Working Sources
- BBC World News RSS
- Wikipedia Current Events
- France24 RSS

## Blocked Sources (451 errors)
- Reuters
- The Guardian

## Verification Rules
- ✅ confirmed = 2+ sources
- ⚠️ single source = 1 source
- ❓ unverified = no sources
